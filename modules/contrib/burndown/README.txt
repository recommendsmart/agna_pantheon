INTRODUCTION
------------

Burndown is a highly configurable project management tool, with functionality
to support both kanban and sprint-based projects.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/burndown

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/search/burndown


REQUIREMENTS
------------

Burndown works with Drupal 8.8.0 and up. It has no other special requirements.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.
   See: https://www.drupal.org/node/895232 for further information.


CONFIGURATION
-------------

No configuration is needed.
