INTRODUCTION
------------

The Field List Details module displays information about fields on various
field management screens within the Drupal UI. It takes into account many
field types within Drupal core and some contributed field types. Other modules
may also provide field details using hook_field_list_details_alter(). See
field_list_details.api.php for further information.

 * For a full description of the module, visit the project page:
   https://drupal.org/project/field_list_details

 * To submit bug reports and feature suggestions, or to track changes:
   https://drupal.org/project/issues/field_list_details

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit:
   https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules
   for further information.

CONFIGURATION
-------------

There is no configuration for this module. Simply enable it and visit your field
management screens.
