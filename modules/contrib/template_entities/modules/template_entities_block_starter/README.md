## Overview

The template entities block starter kit provides a generic block content template type, a block content template vocabulary, and an associated view to bring content block and content block template management within the content area of the site for use by users with no structure or configuration administration permissions.

* Creates a content block template vocabulary.
* Creates a generic content block template type that can be used with any block content type (i.e. custom blocks of any bundle), with additional taxonomy field.
* The template type is configured to place an "Add using Content block template" action button to the supplied content block overview page.
* Additional "templates" tab added to the supplied content block overview page to list all templates of the generic content block template type.
